How does Javascript work?

JavaScript is an interpreted language that is executed in the browser with the help of an engine called the V8 engine. It is a JavaScript engine that is written in C++. At first, JavaScript source code is parsed into a syntax tree which is later interpreted line by line. While being interpreted, the V8 engine uses a compiler called Just In Time (JIT) compiler side by side to execute the code quickly. The V8 engine uses another compiler that optimizes JavaScript code alongside this. Finally, the code is converted into bytecode, code that can be used by computers.

Differences between Local Storage and Session Storage

Local storage and session storage are properties that allow sites to store a collection of keys and values in the browser. These keys and values are used by the browser as objects. Both local storage and session storage have separate storage fields. The main difference between these two is that in session storage, the data is cleared after the browser window is closed. However, in local storage, the data does not expire even after the browser window is closed.
